Inicio do projeto

Este projeto teve como objetivo a criação de um circuito para medir a temperatura durante a fabricação artesanal de cerveja. Utilizamos um Arduino, um sensor LM35, um módulo relê e um display LCD para montar o circuito de medição. Como base para a produção, foi definido o estilo Belgian Blond Ale, para seguirmos como inspiração.
